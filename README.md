# Data-Visualization-Python-Charts-for-Insightss

Dataset Name: World Happiness Report

Tools & Technologies

Python

Pandas

Matplotlib

Jupyter Notebook

Visualizations Created
1️ Bar Chart – Top 10 Happiest Countries

Visualized the top 10 countries based on happiness score

Clearly labeled axes and rotated country names for readability

Highlighted the highest happiness scores

Purpose: Identify top-performing countries and compare their happiness levels.

2️ Line Chart – Happiness Rank vs Happiness Score

Plotted happiness score against happiness rank

Data sorted by rank to ensure correct visualization

Purpose: Show how happiness score decreases as rank increases.

3️ Histogram – Distribution of Happiness Scores

Displayed the distribution of happiness scores across countries

Revealed skewness and spread of data

Purpose: Understand overall distribution and detect possible outliers.

4️ Scatter Plot – GDP per Capita vs Happiness Score

Compared economic prosperity with happiness score

Included gridlines and labels for clarity

Purpose: Analyze correlation between GDP per capita and happiness.
